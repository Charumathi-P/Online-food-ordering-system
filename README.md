# 🍔 Online Food Ordering System
An Online Food Ordering System developed using Java, MySQL, HTML, CSS, and JavaScript. This application enables customers to browse food items, add them to the cart, place orders, and track their order history. It also provides an admin panel for managing food items and customer orders.

## 🚀 Features

### Customer
- User Registration & Login
- Browse Food Menu
- Search Food Items
- Add to Cart
- Update Cart
- Remove Items from Cart
- Place Orders
- View Order History

### Admin
- Admin Login
- Add Food Items
- Update Food Details
- Delete Food Items
- View Customers
- Manage Orders
- Dashboard

## 🛠️ Technologies Used
- Java
- JDBC
- MySQL
- HTML5
- CSS3
- JavaScript
- VS Code

## 🗄️ Database Tables
- Users
- Food
- Cart
- Orders
- Order_Items
- Admin

## ⚙️ Installation
1. Clone the repository
2. Import the MySQL database.
3. Update database credentials.
4. Open the project in VS Code.
5. Run the application.
